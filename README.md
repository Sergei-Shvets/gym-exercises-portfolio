This is a gym exercises application, that created with ReactJS, Material UI and Rapid API. The data for this app received through Rapid API. 
You can find a lot of exercises for improving your body, gain muscles and burn fat. They are separated on muscle groups. You can click any exercise-card and find additional materials, like Youtube-videos and something else.

You can find this app by the following address: gym-exercises-ssl.netlify.app
